## 코드 블록 (Expressive Code)

인라인 문법 강조도 지원됩니다: `console.log("안녕하세요"){:js}`

### 기본 코드 블록

```typescript title="example.ts" showLineNumbers=false
function greet(name: string): string {
  return `안녕하세요, ${name}님!`
}

console.log(greet('아올다'))
```

### Diff 하이라이트 (줄 단위)

`ins`와 `del` 속성으로 추가/삭제된 줄을 표시할 수 있습니다.

```js title="config.js" ins={3} del={2}
const config = {
  theme: 'light',
  theme: 'dark',
}
```

### Diff 하이라이트 (diff 언어)

`diff` 언어와 `lang` 속성을 조합하면 `+`/`-` 접두사로 diff를 표현할 수 있습니다.

```diff lang="js"
- app.configure("api-key-string");
+ await app.configure({ key: "api-key-string", version: "2.0" });

- app.on('event', callback);
+ app.on('event', { handler: callback, options: { once: true } });
```

### 줄 하이라이트 및 접기

```python title="algorithm.py" {2-3} collapse={6-10}
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    # 재귀적으로 피보나치 수를 계산합니다
    # 이 방식은 시간복잡도가 O(2^n)입니다
    # 메모이제이션을 사용하면 O(n)으로 개선할 수 있습니다
    # 아래는 단순 재귀 구현입니다
    # 큰 수에서는 비효율적입니다
    return fibonacci(n - 1) + fibonacci(n - 2)
```

### 키워드 하이라이트 및 주석 블록

키워드를 따옴표로 감싸면 해당 단어가 강조되고, `ins="키워드"`, `del="키워드"`로 키워드 단위 diff도 가능합니다. `{"주석":줄범위}` 문법으로 주석 블록을 추가할 수 있습니다.

```ts title="example.ts" showLineNumbers startLineNumber=100 ins={3} del={4} {5} {"주목할 코드":12-16} collapse={18-22} "awesome" ins="added" del="deleted"
// <- 이 코드블록은 100번째 줄부터 시작합니다!

// 이 줄은 diff 추가로 표시됩니다
// 이 줄은 diff 삭제로 표시됩니다
// 이 줄은 하이라이트됩니다

// "added" 키워드는 초록색으로 표시됩니다
// "deleted" 키워드는 빨간색으로 표시됩니다
// "awesome" 키워드는 파란색으로 표시됩니다

// 주석을 달고 싶은 코드 위에 빈 줄을 삽입하세요 (겹침 방지)

function demonstrateFeatures() {
  console.log('Hello world!')
  return true
}

// 아래 줄들은 접을 수 있습니다
interface HiddenConfig {
  name: string
  age: number
  email: string
}
```

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

행렬도 표현 가능합니다:

$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
\begin{pmatrix}
x \\
y
\end{pmatrix}
=
\begin{pmatrix}
ax + by \\
cx + dy
\end{pmatrix}
$$

### 일반 Callout 예시

<Callout variant="note" title="참고">
  일반적인 정보를 전달할 때 사용합니다.
</Callout>

<Callout variant="tip" title="팁">
  <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>F</kbd> (Windows/Linux) 또는 <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>F</kbd> (Mac)로 코드를 빠르게 포맷할 수 있습니다.

| 동작      | Windows/Linux                                   | Mac                                               |
| --------- | ----------------------------------------------- | ------------------------------------------------- |
| 검색      | <kbd>Ctrl</kbd> + <kbd>F</kbd>                  | <kbd>Cmd</kbd> + <kbd>F</kbd>                     |
| 바꾸기    | <kbd>Ctrl</kbd> + <kbd>H</kbd>                  | <kbd>Cmd</kbd> + <kbd>H</kbd>                     |
| 모두 저장 | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>S</kbd> | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>S</kbd> |

</Callout>

<Callout variant="warning" title="주의">
  이 API는 Internet Explorer에서 **지원되지 않으며**, 구형 브라우저에서 제한적으로 동작합니다.

```js title="polyfill.js"
if (!Object.fromEntries) {
  Object.fromEntries = function (entries) {
    const obj = {}
    for (const [key, value] of entries) {
      obj[key] = value
    }
    return obj
  }
}
```

</Callout>

<Callout variant="danger" title="위험">
이 명령어를 실행하면 현재 디렉토리의 모든 파일이 **영구적으로 삭제**됩니다.

```bash title="danger.sh"
rm -rf ./*
```

이 작업은 되돌릴 수 없습니다.
</Callout>
